# SkillBox Intro Promo Design

## Style Prompt

A 30-second horizontal product-launch short for SkillBox: dark technical, calm but confident, local-first, and security-aware. The visual language should feel like a precise desktop tool for agent builders rather than a generic SaaS landing page. Product screenshots are the hero assets; abstract motion supports the story without inventing unsupported metrics.

## Canvas

- Size: 1920x1080
- Format: 16:9 horizontal
- Duration: 30 seconds
- Background: dark, persistent, technical depth

## Colors

- Deep ink: `#08111F`
- Panel slate: `#111827`
- SkillBox blue: `#0A84FF`
- Trust green: `#2FD17C`
- Soft cyan: `#8FD3FF`
- Off-white: `#F8FAFC`
- Muted text: `#9AA7B8`

## Typography

- Heading voice: `Space Grotesk`, heavy weights, tight but legible.
- Body/UI voice: `IBM Plex Sans`, regular and semibold.
- Data/path voice: `IBM Plex Mono`, tabular numerals.
- Fallback: system UI only if HyperFrames cannot embed the preferred fonts.

## Motion

- Precise UI-like reveals, masked wipes, card slide-ins, small glow pulses.
- Use scene transitions between every major beat.
- Keep screenshot text large enough to read by zooming/cropping instead of showing full UI at tiny scale.
- Choreography should feel measured and intentional, not flashy.

## What Not To Do

- No purple gradient blob background.
- No stock-looking hero imagery.
- No fake metrics or claims not backed by README.
- No unreadably tiny UI labels.
- No jump cuts.
- No crypto/neon overload.
